##### Формат названия миграции
[год `2020`][месяц `03`][день `06`][номер миграции в этом дне `01`][имя миграции]  

Пример: 2020030601_AddSmsFieldsToCalls 
 
`2020` - это год  
`03` - это март  
`06` - день месяца  
`01` - номер миграции     
 `AddSmsFieldsToCalls` - имя миграции связанное с её смыслом
